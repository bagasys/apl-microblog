# # Microblog

## Config

1. Copy `.env.example` jadi `.env`. Bagian penting `.env`:

   ```
     DEFAULT_MODULE=microblog
     ...
     DB_ADAPTER=Phalcon\Db\Adapter\Pdo\Mysql
   ```

2. **BUAT FOLDER KOSONG `apps/cache`**

3. `composer install`

4. Arahin webserver. Kalo udah, buka halaman utama.
